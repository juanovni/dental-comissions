# Proyecto: Dental Social-Clinical CRM (Fase Pro)

## Objetivo
Transformar la gestión de redes sociales de una clínica dental de una tarea de "community management" a una "máquina de conversión clínica y protección de reputación". El sistema une la identidad social (IG/FB) con la identidad clínica (Ficha del Paciente) para medir el ROI real y automatizar la atención inteligente.

## Arquitectura Técnica
- **Backend:** Laravel 11 / PHP 8.3.
- **Admin Panel:** Filament v3 (Customized Workspaces).
- **Base de Datos:** PostgreSQL (con soporte JSONB para metadatos de Meta).
- **IA:** Integración con OpenAI/Claude (Prompt Engineering Clínico).
- **Integraciones:** Meta Graph API (Webhooks) + WhatsApp Business API.

## Alcance Unique Selling Point (USP)
A diferencia de otros CRMs, este sistema conoce el historial clínico del paciente antes de responder en redes sociales y permite rastrear cuánto dinero generó cada post de Instagram mediante un "Handshake" de identidad en WhatsApp.
